## Installation Instructions

### One-Line Summary
Run this command:
```powershell
.\install.ps1 -Profile auto
```

### Step-by-Step
1. Open PowerShell
2. Navigate to project: `cd C:\path\to\pixform`
3. Allow scripts: `Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass`
4. Run installer: `.\install.ps1 -Profile auto`
5. Press Enter when asked about models
6. Wait 15-20 minutes
7. Start app: `.\PIXFORM.bat`
8. Open browser: `http://localhost:8000`
9. Done! 🎉

## Code Changes

**File Modified**: `install.ps1`
**Changes**: 2 small sections (~40 lines)
**Impact**: Zero breaking changes
**Risk**: Zero (fully backward compatible)

### Change 1: Windows OS Detection (Line ~438)
```powershell
if ([Environment]::OSVersion.Platform -eq "Win32NT" -and $SelectedModels["trellis2"]) {
    $SelectedModels["trellis2"] = $false  # Auto-disable on Windows
}
```
